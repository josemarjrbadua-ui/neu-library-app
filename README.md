NEU Library Visitor Management System
A web-based visitor management system developed for the New Era University Library. It allows students and faculty to conveniently check in 
using their institutional email, specify their purpose for visiting, and check out once they leave the premises. The system helps ensure an 
organized and efficient flow of visitors while maintaining accurate records. On the administrative side, it provides tools to monitor real-time 
occupancy, review and manage visit logs, handle user accounts, and post important announcements for library users.

Live site: https://neu-library-app-dece3.web.app/
